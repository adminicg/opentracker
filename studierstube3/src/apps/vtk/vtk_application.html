VRLU Studierstube Sample Application

Setup

Diese Beispielapplikation kann als Ausgangspunkt für Eure eigenen Entwicklungen dienen. Bitte berücksichtig folgende Hinweise :

  1. Kopiere das Verzeichnis vtk/ in ein eigenes um, damit die Entwicklungen von verschiedenen Gruppen sich nicht beeinflussen.
  2. Dann mußt Du die Pfade in start.bat und start_ar.bat entsprechend abändern.
  3. Um eine eigene Application Klasse zu erhalten, kopiere SoSampleKit.h und SoSampleKit.cxx um auf Deinen Klassennamen und ändere den Namen der Klasse in den source files.
  4. Ändere die Files in dem VisualC++ Project und ändere den Namen der erzeugten dll auf Deinen eigenen, etwa whatever_stb.dll.

Dokumentation

VtkKit ist eine einfache Application, die in ein Fenster vorgegebene Geometrie lädt und gewisse Objekte darin mit dem output von verschiedenen Tracker Stations verbindet. Mittels verschiedener tracker config files können diese Objekte entweder über die Tastatur oder mit ARToolkit Markern bewegt werden.

Geometrie die von einer bestimmten Station bewegt werden soll, kann in einem .iv file mit dem Node SoTrackedArtifactKit definiert werden. Ein Beispiel sieht folgendermaßen aus (weitere finden sich in vtk.iv):

SoTrackedArtifactKit {
  station 4  # Nummer der station mit der die Geometrie verbunden wird
  geometry Separator { # Ein Node als geometry, damit wir mehr machen können
    Cube {}            # verwenden wir einen Separator mit Kindern.
  }
}

Für einen kurzen Test starte die Applikation mit start.bat und verwende die stations 0 und 1 um Pen und PIP zu steueren und 4-6 um die Objekte zu bewegen.

Die einzelnen files im Detail :

  • SoSampleKit.h / SoSampleKit.cxx - Definition und Implementation der Applikationsklasse. Diese wird vom Studierstube Hauptprogramm instantiiert und führt die Applikation aus.
  • vtk_stb.la - loadable application module, wird vom Hauptprogramm geladen um die Applikation auszuführen.
  • vtk.iv - Dieses OpenInventor file enthält die Sample Application mit einem eigenen PipSheet und einem Window. Dieses File wird von der Studierstube geladen und ausgeführt.
  • vtktracker.xml - OpenTracker config file, das fünf Stations über Tastatur steuern lässt.
  • artracker.xml - OpenTracker config file, das PIP und Pen über Tastatur und Cube und Sphere über die ARToolkit Marker pi und kappa bewegt.
  • start.bat - startet die Applikation mit Tastatursteuerung.
  • start_ar.bat - startet die Applikation mit kombinierter Tastatur / ARToolkitsteuerung.
