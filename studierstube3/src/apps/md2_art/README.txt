
  md2atart
  ========

Author: Daniel Wagner
        daniel@ims.tuwien.ac.at

This example demonstrates how to use SoMD2NodeKit and ARToolkit
in studierstube. In order to run this example unmodified you
need to have a webcam and an ARToolkit marker called 'hito'.
The application shows an animated md2 mesh (called Ryla) on
the ARToolkit marker (see sample_screenshot.jpg).
