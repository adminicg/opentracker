Tool Scripts

This page documents a couple of usefull scripts that can be found in tools/scripts. Currently the following scripts are documented :
  1. artoolkit2offaxis (Matlab)

artoolkit2offaxis

This Matlab script calculates SoOffAxisCamera parameter that correspond to a given ARToolkit camera calibration. It creates a camera that acts similar to the ARToolkit setup but using the typical Studierstube/OpenInventor coordinate system. The camera eyepoint resides in the origin and looks towards the negative Z axis. Y is up and X to the right. If the tracking is configured as described in ARToolkit vs. Studierstube, the tracking information will match almost perfectly the video background without further registration.

Input:

  • 3x3 camera calibration matrix output by ARToolkit. Note that you should throw away the last column to arrive at the 3x3 matrix. This matrix is output by OpenTracker during startup.
  • size of the video image. This is also output by OpenTracker duning startup.

Output:

  • position of the camera
  • size of the view plane.

The resulting SoOffAxisCameraKit node should look somewhat like this :

SoOffAxisCamera {
  size [size computed by the script]
  position [position computed by the script]
  eyepointPosition 0 0 0
}
