Studierstube Documentation Project

Welcome to the static html pages of the Studierstube documentation. These pages contain general overview documents, HOWTOS and additional information about running studierstube-based systems.

The Studierstube API documentation can be found here. If this link is broken, you have to create your local Studierstube API pages for offline use by running doxygen with the provided scripts (generate.bat or generate.sh in the /studierstube/doc directory). Alternatively, you can use the online pages at http://www.studierstube.org/doc/stbapi/.

If you are a documentation author, please refer to the documentation HOWTO for further information and guidelines.
