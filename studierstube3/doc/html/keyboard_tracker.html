Keyboard Tracker Use

  • You can move a device (pen or pip) with the cursor keys and the numeric +/-
  • Select a device with <P> (pen) and <L> (pip)
  • Select a user with <1>-<6>
  • Select <T> for translation mode and <R> for rotation mode
  • Press <SPACE> for button 0
  • Select speed with <A> (acellerate) and <B> break
  • Multiple keys can be pressed concurrently
Don't forget to press "view all" before using the keyboard tracker, or the camera will behave in strange ways.
