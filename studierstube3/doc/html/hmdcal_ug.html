USER'S GUIDE for HMD-Calibration

by Rainer Splechtna - splechtna@cg.tuwien.ac.at

Starting the application:

The calibration-application (located in directory bin/apps/calibration/)

is started using the workspace application loader.

After the application has been started, the following menu is displayed on the PIP:

1. PIP-Menu:

full calibration:

starts calibration with 8 marker positions per eye (more accurate)

short calibration:

starts calibration with 4 marker positions per eye (quicker)

exit:

exits the application

After choosing full or short calibration, the calibration process starts.

Calibration process:

There are 8(4) calibration points to be registered per eye. The calibration process starts with the left eye.

A virtual marker is displayed on the left display of the HMD. Press the button on the tip of the pen when the virtual marker and the real marker on the pen are perfectly aligned. After pressing the button the next virtual marker is displayed ...

After all 8 (4) positions for one eye are registered, the program performs a test on the sampled data, to determine whether the data is valid or not. If the data is valid, the registration for the right eye begins, which works in the same way as for the left eye. If the data is not in valid boundaries, the calibration process for the left eye has to be repeated.

When the calibration points for the right eye are also registered properly, the following menu is displayed on the PIP:

2. PIP-Menu:

restart calibration:

back to PIP-Menu 1 (all sampled data will be lost)

test calibration (checkbox):

checked: a virtual marker is displayed (3D) , which should be aligned with the real world marker on the pen, showing the quality of the calibration

unchecked: no virtual marker is displayed

keep calibration (checkbox):

checked: new HMD-setting (calibration) is activated

(note: when pressed for the first time, the new HMD-setting is calculated and saved as file named "userXHMD.iv" (where X is the user-number), hence a short pause in the workspace enviroment (1-2 seconds) may occur)

unchecked: the active HMD-setting is the HMD-setting, which was active when the program was started

exit:

exits the application (note: the active HMD-setting stays active)
 

Step by step guide: HMD-calibration in ´workspace´

after starting the calibration-application:

a) choose calibration method

- PIP-Menu 1 is displayed.

user action: press button ´full calibration´ (recommended) or ´short calibration´

b) registration of the left eye:

repeated 8x (for full calibration) or 4x (for short calibration) :

- a virtual marker is displayed on the left display of the HMD

user action: align virtual marker and the real marker on the pen (same center and same size)

user action: press the button on the tip of the pen when the virtual marker and the real marker on the pen are perfectly aligned

c) validation of sampled data for the left eye:

- valid data: registration of the right eye starts (continue with step d)

- no valid data: "repeat calibration (valid cuts = X)... press button" is displayed on the HMD-display

user action: press button on the tip of the pen to restart the calibration for the left eye (continue with step b).

d) registration of the right eye:

repeated 8x (for full calibration) or 4x (for short calibration) :

- a virtual marker is displayed on the right display of the HMD

user action: align virtual marker and the real marker on the pen (same center and same size)

user action: press the button on the tip of the pen when the virtual marker and the real marker on the pen are perfectly aligned

e) validation of sampled data for the right eye:

- valid data: registration completed (continue with step f)

- no valid data: "repeat calibration (valid cuts = X)... press button" is displayed on the HMD-display

user action: press button on the tip of the pen to restart the calibration for the right eye (continue with step d).

f) activate calibrated HMD-setting

- PIP-Menu 2 is displayed

user action: press button ´keep calibration´

- the calibrated HMD-setting is activated and saved as file named "userXHMD.iv" (where X is the user-number)

user action: press button ´exit´

- the application is stopped (the calibrated HMD-setting stays active)

Key Commands:

M -  turn on/off mirror-mode, only useful for video camera calibration (using video background)
         (note: mirror-mode can only be changed before starting the calibration process)
S  -   start 'full calibration' (only before calibration process)
BACKSPACE- repeat registration of previous marker position (only during calibration process)
K - 'keep camera' (only after calibration process)
C -  exit application
