"Versatile Viewer Architecture for Studierstube"

OVERVIEW

The new viewer architecture for Studierstube represents a flexible and extensible implementation.
In its new form the viewer architecture separates the display mode from the camera control mode. The following modes are available in the current implementation:

Display modes:

  • mono (for onscreen display)
  • line interleaved (for iglasses)
  • field interleaved (for glasstron and shutter glasses)
  • dual channel (for onyx + barco doubleprojector with passive polarisation)

Camera control modes:

  • desktop (no head tracking, camera is moved with the mouse)
  • tracked display (for hmds)
  • tracked viewpoint (for projectionsystems, e.g. virtual table)

Any combination of display mode and camera control mode may be used, hence the total number of combinations is 12 (4*3).

Apart from the decoupling of display mode and camera control mode the viewer implementation has the following features, to give the user a flexible architecture:

  • support for X-windows and MS-windows
  • configurable window-size and position
  • support for up to 4 users per workstation (impact channel option!)
  • etc.

 

Possible Hardware for output:

Workstations:

  • SGI Indigo2 Maximum Impact with Impact Channel Option (max. 4 users)
  • Intergraph TDZ2000 Wildcat
  • SGI O2
  • SGI Octane
  • SGI Onyx with 8 channels

Display devices:

  • various monitors
  • Virtual-io iglasses HMD
  • Sony glasstron HMD
  • StereoGraphics CrystalEyes shutter glasses
  • Barco BARON Virtual Table (including shutter glasses)
  • barco doubleprojector with passive polarisation

 

Configuration examples:

The following configurations show some examples of possible useful usage of the new viewer architecture:

mono/desktop:

output: on screen without stereo, no head tracking (control: mouse, keyboard)
usage: for control purposes and demos for larger groups, …
supported hardware: indigo2, o2, octane, onyx, intergraph (Windows-NT)

field interleaved/desktop:

output: on screen or virtual table with stereo (using shutter glasses), no head tracking (control: mouse, keyboard)
usage: stereo demos for groups, …
supported hardware: indigo2, octane, o2

dual channel/desktop

output: on large projection screens with stereo, no head tracking (control: mouse, keyboard)
usage: stereo demos for groups, …
supported hardware: onyx

line interleaved/tracked display

output: iglasses (stereo), head tracking
usage: standard-studierstube
supported hardware: indigo2, o2, intergraph

field interleaved/tracked display

output: glasstron (stereo), head tracking
usage: standard-studierstube
supported hardware: indigo2, intergraph

field interleaved/tracked head

output: on screen or virtual table with stereo (using shutter glasses), head tracking
usage: standard-studierstube
supported hardware: indigo2, octane, o2

dual channel/tracked head

output: on large projection screens with stereo, head tracking
usage: standard-studierstube
supported hardware: onyx
