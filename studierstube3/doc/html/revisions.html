Studierstube Revision History and Roadmap

Studierstube 1.0

Release date: September 1998
  • First release of Studierstube code
  • Intermediate release at TUW and CRCG
  • Rudimentary support for platform abstraction (HMD, Virtual Table)

Studierstube 1.1

Release date: November 1998
  • Release used for application development at CRCG Providence, IGD Darmstadt, GRIS Tübingen
  • Reworked directory structure
  • Many internal restructurings
  • Somewhat improved platform abstraction
  • Begin of documentation

Studierstube 1.2

Release date: October 1999
  • Intermediate internal release at TUW
  • Added Workspace extentions, multi-tasking
  • Added 3D Window extentions
  • Redesigned directory structure (again)
  • Various bug fixes, cleanups
  • Improved tracker server
    • still compatible with old network protocol
    • Extended range transmitter support
    • Correct built-in tracker filter handling
  • Full support for NT and IRIX
  • Full support for Open Inventor 2.52

Studierstube 1.3

Release date: March 2000
  • New generalized viewer classes (support for HMD, Virtual Table, Projection Wall, desktop, ...)
  • Re-designed event system (Stb3DEventGenerator, new SoTrakEngine)
  • New tracker server (inclusing new network protocol and prediction)
  • New tracker tree subsystem allows versatile configuration of trackers
  • Calibration applet (implementation: Rainer Splechtna)

Studierstube 1.99

Release date: June 2000
Standard features:
  • Improved performance
  • Numerous bug fixes and improvements over 1.3
  • Storyboarding application
  • AVS Interface 2.0
Experimental features:
  • Distributed Open Inventor (DIV) allows basic distribution of graphics (experimental feature)
  • Distributed Workspace allows to run Studierstube on multiple workstations
  • Context and places management with Distribution Manager and Session Manager components
  • Application migration

Studierstube 2.0

Release date: December 2000
  • Open source version (based on SGI  OIV, SoQt etc.)
  • Distributed Open Inventor 2.0 based on SGI OIV source (experimental feature)
  • Redesigned resource manager (user kit, pip kit, pen kit)
  • Introduction of user kits (scriptable user and viewer configuration)

Studierstube 2.01

Release date: March 2001

For a while, we have been running our cvs repository to manage the Studierstube code base. Therefore, we only highlight here some highlights of version 2.01. In the future, please watch the Studierstube commit log for updates.

  • cvs repository for Studierstube developers (Gerhard Reitmayr)
  • Documentation based on Doxygen, linked with old documentation (Dieter Schmalstieg)
  • Distributed Studierstube (Gerd Hesina)
  • Studierstube runs under Linux (Jan Prikryl)
  • New Build System (Makefiles) for SGI and Linux (Jan Prikryl)
  • New Build System (single Visual Studio project file) for Win32 (Jan Prikryl)
  • Gesture recognition (Benjamin Schmit)
  • Chess applet for 2 player (Dieter Schmalstieg)
  • Construct3D application (Hannes Kaufmann)
  • Touch widgets for buttonless interaction (Jan Prikryl)

Studierstube 2.1

Expected release date: Early 2001
  • OpenTracker: Integration of new tracking framework
  • Distributed Workspace fully functional and tested

Studierstube 3.0

Expected release date:  Late 2001
  • Mobile Studierstube
  • Automatic joining and leaving of participants
  • Instant collaboration by wireless networking
  • Managing of mobile contexts and places
