Studierstube 3.3 Distribution

Studierstube 3.3
Oktober 2004

This CD contains the necessary software packages needed to setup a Studierstube development environment for Windows. It contains pre build binaries and allows you to start developing almost immediately. It also contains all necessary libraries and sources to compile the distribution on Linux.

Content

A short overview of the contents of this CD:

  • ACE - Adaptive Communications Environment, a platform abstraction library used by OpenTracker
  • ARVideoLib - an ARToolKit version for Windows using an optimized DirectShow interface library. This version is used by OpenTracker.
  • Coin3D-2 - Coin3D, Open Inventor compatible 3D rendering library
  • contrib - patches applied to Coin3D
  • docs - OpenInventor Manuals and help files, other Studierstube related documentation
  • examples - a simple blue print for a Studierstube application
  • fmodapi371win32 - FMOD sound library
  • opentracker - Tracking library that supports a variety of input devices
  • pivy - Python Open Inventor binding that supports a script node for Python programming
  • studierstube - The Studierstube itself.
  • wintab - graphic tablet drivers, used by OpenTracker
  • xerces - XML Parser for OpenTracker

All packages are pre build and ready to run. However, there is some installation necessary to put all pieces at the right places.

Installation

Install the CD manually be executing the following steps: 

  1. Copy or extract the CD to your hard drive:

    Copy the contents of the CD to a directory on your hard drive. If the CD contains a ZIP file extract it to your hard drive. For example to c:\stb.

  2. Start the installer Studierstube.msi

  3. That's it you are ready to go !

If something goes wrong with the installer you can execute the following steps manually:

  1. Environment variables :

    To allow all programs to find the necessary DLLs, you need to set a number of environment variables. It is practical to do this on a per-user basis to avoid conflicting settings by different users. Set the following variables :

    • STBROOT to the directory studierstube (e.g. c:\stb\studierstube)
    • ACEROOT to the directory ACE (e.g. c:\stb\ACE)
    • ARTOOLKITROOT to the directory arvideolib (e.g. c:\stb\arvideolib)
    • BOOST to the directory boost_1_31_0 (e.g. c:\stb\boost_1_31_0)
    • CAL3D to the directory cal3d (e.g. c:\stb\cal3d)
    • COINROOT to the directory Coin3D-2 (e.g. c:\stb\Coin3D-2)
    • FMODROOT to the directory fmodapi371win32 (e.g. c:\stb\fmodapi371win32)
    • OTROOT to the directory opentracker (e.g. c:\stb\opentracker)
    • XERCESCROOT to the directory xerces (e.g. c:\stb\xerces)
    • Append the following to the PATH variable :
      %STBROOT%\lib;%ACEROOT%\lib;%XERCESCROOT%\bin;%OTROOT%\bin;%COINROOT%\bin
  2. Install DirectShow Renderer

    Run regsvr32.exe on the file ARVideoLib\bin.vc70\DsRenderer.ax. This program will register a DirectShow renderer control for optimal video capture. It can be typically found in %SYSTEM32%.

Test run

To give Studierstube a quick test run, go to the directory studierstube\bin\workspace and start the following command :
workspaced -a animal.iv

This will open a simple desktop Studierstube window presenting you with a pen, a PIP and a simple example application. You can use the OpenInventor GUI elements to manipulate the view of the scene. The pen and the PIP are controlled by the following list of keys :

Y+ cursor up
Y- cursor down
X+ cursor right
X- cursor left
Z+ page up
Z- page down
rotate around X+ e
rotate around X- d
rotate around Y+ r
rotate around Y- f
rotate around Z+ t
rotate around Z- g
reset w
button 0 space bar
button 1 m
switch to Pen 0
switch to PIP 1

Development Environment

Our main development environment is Microsoft Visual C++ .Net 2003. The project files are configured for this compiler. You can use another compiler, however you will have to configure your build environment for yourself.

Examples

A small number of very simple examples are in the directory examples\.

  • helloworld.iv - a simple hello world text in 3D and on the PIP.
  • select.iv - a button on the PIP to switch colors.
  • nurbscurve.iv - interaction and widgets together.
  • sample\ - skeleton C++ application.

Examples are started from the directory by calling workspaced with the parameter -a :
..\studierstube\bin\workspace\workspaced -a helloworld.iv

Notes

Coin3D source and patches

The version of Coin3D distributed on this CD contains a number of patches to work around problems in the Coin3D library and extend some functionality. It is compiled from the Coin-2 CVS repository from 27 of September 2004. The patches can be found in the contrib directory on the CD.

Runtime library versions (Debug/Release)

Studierstube and all the necessary support libraries are compiled and linked against both the debug and release versions of the VC++ runtime library. They can be distinguished by the typical 'd' postfix for debug versions. The project configurations are called Debug Coin and Release Coin.

Running Studierstube under Unix

See the Unix Build Instructions. You have to download the necessary source packages as required.

Updates

The Studierstube and the OpenTracker directories are checked out of the Studierstube Subversion directory with the user anonymous. This allows you to update the code base to the current version in the CVS at any time. The password for the user anonymous is stbsvn. The repository is https://www.studierstube.org/svn/studierstube/trunk. To browse it online go to http://www.studierstube.org/viewcvs/viewcvs.cgi/trunk/
